# Web Paint 

A simple analog of Paint directly in the browser with the ability to save a drawing to a computer.

---

## Features

- 6 colors to choose from
- Choosing the brush thickness
- Saving the finished drawing to a computer (800x800 px)
- Clearing the canvas in one click

---

## Built with

- HTML
- CSS
- JavaScript
